# She Can Foundation - Frontend Internship Project

## Project Overview

This project was developed as part of the Frontend Development Internship selection task for She Can Foundation.

The website is a responsive single-page application designed to present the organization's mission, vision, and commitment to empowering students through education, awareness, and community-driven initiatives.

## Features

- Responsive Design
- Modern User Interface
- About Section
- Interactive Buttons
- Smooth Scrolling Navigation
- Dark Mode Toggle
- Hover Effects & Animations
- Mobile-Friendly Layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
